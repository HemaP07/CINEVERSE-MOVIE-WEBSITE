# CINEVERSE-MOVIE-WEBSITE
I built 🎬 CineVerse – A Premium Movie Search Web App using HTML, CSS, Bootstrap, and JavaScript.

✨ Features

🎬 Search movies using the OMDb API
⭐ IMDb ratings and movie details
🔥 Trending Movies section
🏆 Top Rated Movies section
🎭 Browse by Genre
❤️ Add & remove favorite movies (Local Storage)
🎥 Watch Trailer button
🔍 Live search suggestions
🌙 Dark Mode
📱 Fully Responsive Design
💎 Modern Glassmorphism UI

🛠️ Technologies Used
HTML5
CSS3
Bootstrap 5
JavaScript (ES6)
Fetch API
Local Storage
OMDb API
